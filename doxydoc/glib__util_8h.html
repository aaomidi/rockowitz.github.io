gpointer* g_list_to_g_array ( GList *  glist,
guint *  length 
)

Converts a doubly linked list of pointers into a null-terminated array of pointers.

Parameters
glistpointer to doubly linked list
lengthwhere to return number of items in the allocated array, (not including the final NULL terminating entry
Returns
pointer to the newly allocated gpointer array
Remarks
The pointers in the linked list are copied to the newly allocated array. The data pointed to is not duplicated.
This function is needed because glib function g_hash_table_get_keys_as_array() does not exist in glib versions less than 2.40
